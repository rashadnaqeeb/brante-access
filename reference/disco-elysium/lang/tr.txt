# Register: standard written Turkish, informal "sen", terse neutral screen-reader style,
# matching the game's own Turkish localization (Buradasın, tekrar deneyebilirsin, bas).
# Game vocabulary reused: check = sınama (beyaz/kırmızı), skill = beceri, skill point =
# beceri puanı, ability = nitelik, thought = düşünce, thought cabinet = Düşünce Diyarı,
# forget = unut, health = sağlık, morale = moral, inventory = envanter, journal = günlük,
# character sheet = karakter sayfası, task = görev, signature skill = kişisel beceri,
# experience = tecrübe, research time = araştırma süresi, locked = kilitli, keys =
# anahtarlar, bullets = mermi, clothing = kıyafetler, item = eşya, currency = réal,
# quicktravel = hızlı geçiş, pawnshop = ikinci el dükkânı, collage = kolaj.
# Place names taken from the game's own Turkish text: secretary's office = sekreterin
# ofisi, union office = sendika ofisi, cargo container = kargo konteyneri, buoy =
# şamandıra, Fisherman Shacks = Balıkçı Barakaları, waterfront = Martinaise Kıyıları.
# Container words name the props each dev token actually matches: can = içecek kutusu
# (deposit-tare drink cans; the game's teneke kutu is canned food), pot = saksı (the one
# flower pot), locker = çelik dolap (the
# sea-fortress steel lockers) against cabinet = dolap (the secretary's office cabinet),
# crate = kasa against safe = para kasası. A lootable container is a kap; konteyner alone
# is a freight container. Turkish collapses door/gate and stairs/ladder onto kapı and
# merdiven, so the second of each pair is marked (büyük kapı, el merdiveni) to keep the
# distinction a blind player needs; geçit is the game's word for a passage, not a gate.
# Turkish has no grammatical gender, so a word spoken after a noun the game picks (an
# item, a category, a health bar) needs no agreement: boş, yeni, yok, kritik seviyede.
# Turkish keeps a noun singular after a numeral, so every counted phrase has one form.
